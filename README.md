## 🎯 What Is SkillLumen?

Most internship platforms show you a list and call it a day.

**SkillLumen** is different. It maps your full journey — from student to industry professional — across 8 domains, with structured task tiers, real mentorship, and a certificate that actually means something.

```
You Apply → Get Placed in a Domain → Easy Task → Medium Task → Hard Task → Certificate
```

No fluff. No filler. Just real work that builds a real portfolio.

## 🛠️ Tech Stack

Built with zero frameworks — pure control over every pixel.

| Layer | Choice | Why |
|---|---|---|
| Structure | HTML5 Semantic | SEO + accessibility |
| Styling | CSS3 Custom Grid/Flex | No Bootstrap bloat |
| Logic | Vanilla JS ES12+ | Direct DOM mastery |
| Assets | Optimized SVG/WebP | Sub-400ms load |

> "I didn't use React or Vue — because a real architect understands the DOM first."

## 🚀 Quick Start

```bash
# Clone
git clone https://github.com/MaazShahid1111/skilllumen.git
cd skilllumen

# No npm needed — just open it
python -m http.server 8080
# → Open http://localhost:8080
```

**Adding a new domain?** Open `index.html`, find `domainData` in the `<script>` tag, add your entry — the card and modal generate automatically.
